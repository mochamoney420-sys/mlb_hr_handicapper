WHAT GETS LEARNED
========================================================================

MISSED HR (Most Important Learning):
  When: Model predicted low (e.g., 8%) but batter hit HR
  System: Flags as "MISSED_HR"
  Learning: Boosts this player's training data 3.0x
  Effect: Tomorrow's model more aggressive with this batter type
  Example: Judge 8%→12% → 18% as model learns

ACCURATE PREDICTION (Reinforcement):
  When: Model predicted correctly (e.g., 20% and HR happened)
  System: Flags as "PREDICTED"
  Learning: Boosts this training row 1.5x
  Effect: Model learns confidence in this signal was right

PITCHER VULNERABILITY (Real-time):
  When: Pitcher gives up multiple HRs in a day
  System: Updates recent HR allowed rate automatically
  Learning: Any batter vs this pitcher gets edge boost tomorrow
  Example: Pitcher allows 3 HR → Recent rate 0.12 → Matchups flagged

BATTER HOT STREAK (Pattern Recognition):
  When: Batter gets multiple HRs in short time
  System: Exit velocity and barrel rate in rolling windows increase
  Learning: Model sees elevated metrics, predicts higher
  Example: 2 HRs in 3 days → Barrel rate 18% → Prob +3%

WEATHER/PARK EFFECTS (Contextual):
  When: Day with specific conditions → Multiple HRs
  System: Records temp, wind, park factor for each HR
  Learning: Future similar conditions recognized as HR-friendly
